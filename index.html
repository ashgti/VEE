Virtual Embedded Enviorment
Author:
John Harrison <ash.gti@gmail.com>

The Virtual Emulated Environment or VEE is an emulation system for learning about embedded systems and embedded system related problems.

This is primarily a teaching tool, but could be used as a test platform for some Arduino Code.

The VEE system targets the Atmega328p processor for the Arduino development enviorment. VEE implements some of the hardware details of the Atmega328p as well as most of the Arduino API's that sit on top of the Atmega processors.

The emulator has four major components. The Arduino API, the Atmega328p Hardware Abstractions, Interrupt Dispatcher and a python interface for controlling the system.
