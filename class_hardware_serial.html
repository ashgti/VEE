Detailed Description

Hardware Serial represents the a basic hardware RS-232 Serial port.

In practice, the only hardware serial port is contained on a special set of pins that the hardware supports. This class will work with any set of pins you pass it, so this is a specific divergence from the actual hardware implementation.
